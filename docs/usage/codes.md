# 机器码
> 用来展示各种标准的条形码、二维码

## Code Apron
Code Apron 是与 Apron Design 团队开发的展示各种机器码的独立项目，也是 Apron Design 的分支项目。

虽然 Code Apron 是 Apron Design 的分支项目，但这个项目几乎没有样式设置，你也可以在其他组件库为 base 开发的项目中使用它。

图标库在 React 和 Vue3 上可以直接通过 `npm` 安装使用，在其他场景下需要下载 svg 来使用。

## 安装和使用
请参照 React 组件库和 Vue3 组件库文档中有关二维码（QRCode）、二维码（Pdf417）和条形码（Barcode）部分的详细解说。
