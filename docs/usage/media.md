# 音视频媒体工具
> 音视频播放器、拉流工具。

## Media Apron

Media Apron 是与 Apron Design 最佳搭配使用的、用来渲染媒体、流的独立项目，也是 Apron Design 的分支项目。

虽然 Media Apron 是 Apron Design 的分支项目，你也可以在其他组件库为 base 开发的项目中使用它，但需要自己污染样式。

图标库在 React 和 Vue3 上可以直接通过 `npm` 安装使用，在其他场景下需要下载 svg 来使用。

## 安装和使用
请参照 React 组件库和 Vue3 组件库文档中有关视频播放器（Video）、音频播放器（Audio）等部分的详细解说。
