# 推荐搭配

## 走马灯
推荐使用 **Swiper** 项目创建功能强大的走马灯。

* [Swiper](https://swiperjs.com/)
* [Swiper npm](https://www.npmjs.com/package/swiper)

## 很酷的动效
推荐使用 **ReactBits** 和 **VueBits** 项目创建酷炫的动效。

本文档网站大量地使用了 ReactBits 创建动画。

* [ReactBits](https://reactbits.dev/)
* [VueBits](https://vue-bits.dev/)

## 缓动动效
推荐使用 **AOS** 创建缓动动效，可以让你的页面在滑动到容器所在位置时添加一个动效。

* [AOS](https://michalsnik.github.io/aos/)
* [AOS Github](https://github.com/michalsnik/aos)

## 交互动效
推荐使用 **GSAP** 创建交互式动效，你也可以轻松地做出苹果官网、大疆产品展示的动效。

* [GSAP](https://gsap.com/)
* [GSAP Github](https://github.com/greensock/GSAP)
